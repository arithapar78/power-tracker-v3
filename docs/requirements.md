# Power Tracker v3 MVP Requirements

## Goal
Rebuild Power Tracker from scratch with only the minimum required features.

## Must Have
- Chrome extension popup
- Prompt generator
- Frontend event tracking
- Backend database integration
- Basic analytics summary

## Not In Scope
- Premium access code
- Full analytics dashboard
- AI model detection
- OODA agent system
- Pattern recognition
- Website integration
- Export/import tools

## First Milestone
A user can:
1. open the extension popup
2. enter a prompt
3. optimize the prompt
4. save the event to the backend
5. see a simple success response
